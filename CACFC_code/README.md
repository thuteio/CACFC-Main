# Consistency-Guided Anchor in Multi-view Subspace Clustering with Prior-Aware Tensor Norm
### Run the main program

To run the main script, simply execute:
main.m --dataset=wiki; 3sources;  webkb; youtube; prokaryotic; reuters3.

### data
Six multi-view datasets:
wiki; 3sources;  webkb; youtube; prokaryotic; reuters3.

CACFC clustering-related programs.
